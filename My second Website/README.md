# Lone Cars — simple static car sales site

## Local (VS Code)
1. Open folder in VS Code.
2. Ensure files:
   - index.html
   - styles.css
   - script.js
   - sitemap.xml
   - robots.txt
   - images/
3. Open index.html in browser or use Live Server extension.

## Publish (quick)
- GitHub Pages:
  1. Push repository to GitHub.
  2. Settings → Pages → choose branch `main` and folder `/` → Save.
  3. Visit https://<username>.github.io/<repo>/

- Netlify / Vercel: drag & drop the folder or connect repo — automatic deploy.

## Notes
- Replace images in /images/ with real photos.
- Replace `yourdomain.com` in sitemap.xml and meta tags with your domain before submitting to Google Search Console.
